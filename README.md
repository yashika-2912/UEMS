**Project Overview**

The University Event Management System enables administrators to manage events and allows students to register for events through a secure and user-friendly platform. The system minimizes manual effort, prevents duplicate registrations, and maintains structured event records using a relational database.

**Features**

**Authentication & Authorization**

Secure login and registration system

Role-based access for Admin and Student

 **Admin Functionalities**

Create and manage events

View registered students for each event

Track event participation

** Student Functionalities**

Register and log in securely

View available events

Register for events (duplicate registrations prevented)

 **Database Management**

Structured relational database using SQLAlchemy ORM

Separate tables for users, events, and registrations
